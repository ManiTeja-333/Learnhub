Data Analytics

Instructor: Alex

Duration: 6 Weeks

This course teaches you how to analyze and interpret data to make informed decisions. You will learn to work with datasets using tools like Excel and Python, perform data cleaning, and create meaningful visualizations to communicate insights effectively.

Course Curriculum

  • Introduction to Data Analytics
  • Data Collection & Cleaning
  • Excel for Data Analysis
  • Python Basics for Data
  • Data Visualization Techniques
  • Real-world Data Projects
